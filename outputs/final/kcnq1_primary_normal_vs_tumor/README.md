# KCNQ1 Lauren full-group output

This folder contains the non-downsampled KCNQ1 comparison used as input for the downsampling analysis.

## Groups

- `Primary normal epithelial`: epithelial candidate cells from `Primary_Normal` samples.
- `Tumor intestinal patient`: epithelial candidate cells from `Primary_Tumor` samples with intestinal Lauren type.
- `Tumor diffuse patient`: epithelial candidate cells from `Primary_Tumor` samples with diffuse Lauren type.

## Files

- `KCNQ1_primary_normal_vs_tumor_comparison.pdf`: full-group visualization.
- `KCNQ1_primary_normal_vs_tumor_comparison.xlsx`: summary and tests.
- `kcnq1_primary_normal_vs_lauren_tumor_cell_level.tsv`: cell-level table with KCNQ1 raw counts and normalized expression.
- `kcnq1_primary_normal_vs_lauren_tumor_summary.tsv`: group-level summary.
- `kcnq1_primary_normal_vs_lauren_tumor_tests.tsv`: exploratory pooled-cell Fisher and Wilcoxon tests.
- `kcnq1_primary_normal_vs_lauren_tumor_by_patient.tsv`: patient-level summaries.
- `kcnq1_matched_normal_tumor_pairs.tsv`: five matched normal-tumor patient pairs.
- `kcnq1_matched_normal_tumor_paired_tests.tsv`: exact paired Wilcoxon tests.
- `kcnq1_tumor_lauren_patient_tests.tsv`: intestinal-versus-diffuse patient-level tests.
